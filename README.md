# Sistema para Controle de Atendimento

## Descrição do Projeto

Este é um projeto de Sistema de controle de atendimento para laboratórios médicos desenvolvido em Ionic/Angular, implementando um sistema de filas com priorização inteligente e gerenciamento de senhas. 

Desenvolvido com **Ionic Framework** e **Angular**, com foco em soluções para a área médica ou laboratorial.

## Tecnologias Utilizadas

O projeto é construído sobre uma pilha de tecnologias modernas para desenvolvimento de aplicações híbridas:

| Tecnologia | Versão (Inferred) | Uso |
| :--- | :--- | :--- |
| **Ionic Framework** | Mais recente | Interface de usuário e componentes móveis. |
| **Angular** | Mais recente | Estrutura de desenvolvimento front-end. |
| **TypeScript** | Mais recente | Linguagem de programação. |
| **Capacitor** | Mais recente | Runtime nativo para acesso a APIs de dispositivos. |
| **Sass/SCSS** | - | Pré-processador CSS (conforme `angular.json`). |

*Nota: As versões exatas das dependências não puderam ser verificadas devido à ausência do arquivo `package.json` no diretório raiz. As instruções de instalação pressupõem o uso de um gerenciador de pacotes padrão (npm).*

## Pré-requisitos

Para executar este projeto localmente, você precisará ter instalado:

*   [Node.js](https://nodejs.org/) (versão LTS recomendada)
*   [npm](https://www.npmjs.com/) (ou outro gerenciador de pacotes como Yarn ou pnpm)
*   [Ionic CLI](https://ionicframework.com/docs/cli)
*   [Angular CLI](https://angular.dev/tools/cli)

Instale o Ionic CLI globalmente:
```bash
npm install -g @ionic/cli
```
Instale o Angular CLI globalmente:
```bash
npm install -g @angular/cli
```
## Instalação e Execução

Siga os passos abaixo para configurar e executar o projeto:

1.  **Instale as dependências:**
    ```bash
    npm install
    ```

2.  **Execute a aplicação no navegador (modo de desenvolvimento):**
    ```bash
    ng serve
    ```
    O aplicativo será aberto automaticamente no seu navegador padrão em `http://localhost:4200/`.

3. **Pasta do Projeto:**

   Abrir o Projeto na pasta `medlab-ionic` para a sua Execução.
